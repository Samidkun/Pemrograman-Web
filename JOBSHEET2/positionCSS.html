Position: static;

Position Static Digunakan Untuk Mengatur Element Menjadi Static Secara Default. Elemen Akan Mengikuti Posisi Normal Secara Default, Elemen Tidak Dipengaruhi Oleh Properti Seperti Top,Bottom,Left Dan Right.:

Elemen Ini Diatur Menggunakan Posisi:static;




Position: relative;

Sebuah element yang renggunakan position relative akan tertetak pada posisi normal. Mengatur properti atas. kanan. bawah, dan kiri dari elemen yang diposisikan dengan "relatif" akan membuatnya Jauh dari posisi norotnya. Konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan Oleh tersebut. :

Elemen Ini Diatur Menggunakan Posisi : relative;



position: fixed;

Sebuah element HTML yang di setting dengan position fixed akan memiliki Sifat tetap, tanpa ada perubahan bahkan jika halaman website scroll. Berlraku pengaturan left, bottom. top dan right pada position

Elemen Ini Diatur Menggunakan Posisi: fixed;















Position: Absolute

Element HTML yanq position absolute akan diposisikan retatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap Layar secara normal.

Elemen Ini Diatur Menggunakan Posisi: relative;
Elemen Ini Diatur Menggunakan Posisi: absolute
